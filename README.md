Задание 5 - Дробь
Созданные дроби:

f1 = 1/3

f2 = 2/3

f3 = 3/4

f4 = 5/6

Арифметические операции:

Сложение: 1/3 + 2/3 = 1/1 (автоматическое сокращение до 1)

Вычитание: 3/4 - 5/6 = -1/12 (корректный знак)

Умножение: 1/3 × 2/3 = 2/9

Деление: 3/4 ÷ 5/6 = 9/10

Операции с целыми числами:

1/3 + 2 = 7/3

2/3 × 3 = 2/1 (сокращение до 2)

3/4 - 1 = -1/4

5/6 ÷ 2 = 5/12

Цепочка операций:
f1.Sum(f2).Div(f3).Minus(5) → (1/3 + 2/3) ÷ 3/4 - 5 = -11/3
